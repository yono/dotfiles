# dotfiles

## Dependencies

* Homebrew

## Suportted software

* zsh
* vim

## Initialize

```bash
$ git clone git@github.com:yono/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ sh ./init.sh
```

## Update

```bash
$ cd ~/.dotfiles
$ git pull
```

## Local Setting on zsh

```bash
$ vi $HOME/~/.zshrc.local
```
